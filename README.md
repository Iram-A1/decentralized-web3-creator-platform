
# Decentralized Web3 Creator Platform

**Course:** DEVP-600 — Business Innovation with DevOps  
**Institution:** Saskatchewan Polytechnic  
**Group:** Iram Akram & Harjinder Kaur  
**Instructor:** Yongchang He  

## Project Overview
A blockchain-based creator monetization platform deployed on AWS through a complete DevOps pipeline. Creators can publish digital content as NFTs, receive direct ETH tips, and earn ERC-20 token rewards — all without intermediaries.
